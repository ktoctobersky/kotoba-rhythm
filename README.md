# ことばりずむ（Kotoba Rhythm）

**ことばりずむ** は、幼児向けの「ことば × おと × パズル」の知育ウェブアプリです。  
ブラウザだけで動作し、ひらがなを **耳で聞いて・目で見て・手で動かして** 学べることを目指しています。

- 🗣️ **よむモード**：ひらがなが光ってはねながら 1文字ずつ・単語として読まれる
- 🧩 **ぱずるモード**：絵と音声を手がかりに、ひらがなチップをドラッグして並べる

iPad / iPhone / PC のブラウザで動作することを想定しています。

## 機能概要

### 🗣️ よむモード

1. 画面中央に、ことばを構成する **ひらがなブロック** が並ぶ  
2. 下の「よむ」ボタンを押すと：
   - 1文字ずつブロックが **光って少しジャンプ** しながら、その音をゆっくり読み上げる  
   - その後、テンポよく続けて読み上げて「ことば」として聞かせる  
   - 対応する絵文字（🐶 など）が下に表示される  

👉 「この音は、この文字」という対応を視覚的に理解することが目的です。

---

### 🧩 ぱずるモード

1. 上に **絵文字（例：🐶）＋「なにかな？」** が表示される  
   - ことば（「いぬ」など）は表示しません（ネタバレ防止）
2. 絵と音声（ことばを読み上げる）をヒントにしながら：
   - 真ん中に **空のスロット**（例：3文字ぶんの箱）  
   - 下に **ひらがなチップ** が並ぶ  
     - 正解の文字（もら）  
     - ＋少しだけ **ハズレのひらがな** が混ざっている
3. 子どもは、チップを **ドラッグしてスロットに入れる**：
   - クリック（タップ）だけでは何も起きない  
   - 指で「つかんで→はこにドロップ」する操作に統一
4. すべてのスロットが埋まると自動で判定：
   - ✅ 正解：  
     - 「ぴんぽーん！ せいかい！」と表示  
     - 音声でも「せいかい！」  
   - ❌ 不正解：  
     - 「ざんねん… せいかいは『◯◯』」と表示  
     - スロットを **正しい並び** で上書き表示  
     - 音声でも「ざんねん。せいかいは ◯◯」
